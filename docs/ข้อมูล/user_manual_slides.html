บทบาทผู้ใช้งาน 3 ประเภท

👤

User (ผู้ใช้ทั่วไป)

  • ประเมินตนเอง
  • ประเมินผู้อื่นตามที่ได้รับมอบหมาย
  • ดูสถานะการประเมิน
  • ทำแบบสำรวจความพึงพอใจ
  • แก้ไขโปรไฟล์ส่วนตัว
👑

Admin (ผู้ดูแลระบบ)

  • จัดการผู้ใช้งาน
  • จัดการโครงสร้างองค์กร
  • สร้าง/จัดการแบบประเมิน
  • มอบหมายการประเมิน
  • จัดการ Access Code
  • ดูรายงาน + Export
🌐

External (ผู้ประเมินภายนอก)

  • เข้าระบบด้วย Access Code
  • ยืนยันตัวตนและองค์กร
  • ทำแบบประเมินตามที่กำหนด
  • ประเมินได้ครั้งเดียว/Code

ฟังก์ชัน Admin — จัดการข้อมูลหลัก

จัดการผู้ใช้ (/admin/users)

  • เพิ่ม / แก้ไข / ลบผู้ใช้
  • กำหนดบทบาท (user / admin)
  • กำหนดระดับ (grade)
  • กำหนดสังกัด (สายงาน/หน่วยงาน/ตำแหน่ง/ฝ่าย)
  • Inline create สำหรับสร้างหน่วยงานใหม่ในฟอร์ม

จัดการโครงสร้างองค์กร

ระดับURL
สายงาน/admin/divisions
หน่วยงาน/admin/departments
ตำแหน่ง/admin/positions
ฝ่าย/admin/factions

โครงสร้างองค์กร

สายงาน (Division) └── หน่วยงาน (Department) └── ตำแหน่ง (Position) ฝ่าย (Faction) ↑ อิสระจากลำดับชั้น (กลุ่มข้ามสายงาน)
ระวัง: การลบสายงานจะลบหน่วยงานและตำแหน่งที่เกี่ยวข้องทั้งหมด (Cascade Delete)

ฟังก์ชัน Admin — มอบหมายและ Access Code

มอบหมายการประเมิน (/admin/assignments)

  • สร้างรายบุคคล: เลือกผู้ประเมิน → ผู้ถูกประเมิน → มุม → ปีงบ
  • สร้างจำนวนมาก (Bulk): สร้างหลายรายการพร้อมกัน
  • แก้ไข/ลบ: ทั้งรายบุคคลและ Bulk Delete
  • Analytics: สถิติภาพรวม
  • Export: ส่งออกข้อมูลการมอบหมาย

มุมการประเมิน

มุมความหมาย
topหัวหน้า → ลูกน้อง
bottomลูกน้อง → หัวหน้า
leftเพื่อนระดับเดียวกัน
rightข้ามสายงาน / ภายนอก

จัดการ Access Code (/admin/access-codes)

สำหรับผู้ประเมินภายนอก

  • Generate: สร้าง Code ใหม่ — เลือกองค์กร, ผู้ถูกประเมิน, แบบประเมิน
  • Revoke: เพิกถอน Code
  • Regenerate: สร้างใหม่ทดแทน
  • Export: ส่งออกรายการ Code
  • Print Cards: พิมพ์บัตร Code

จัดการองค์กรภายนอก

/admin/external-organizations
สร้าง / แก้ไข / ลบ / เปิด-ปิดสถานะ
ใช้ร่วมกับ Access Code
